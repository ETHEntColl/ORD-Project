## GS-Viewer-web
Gaussian Splat viewer for viewing one GS model. Based on the GaussianSplat3D viewer by mkkellogg:
https://github.com/mkkellogg/GaussianSplats3D

Supported data types: .ply, .splat, .ksplat

Not supported: .glb, ...
